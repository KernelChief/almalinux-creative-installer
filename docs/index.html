Installation

Ready in minutes,
v1.0.14 Stable (GTK)

Download the RPM for your AlmaLinux version, install it with DNF, and you're ready to create. No manual configuration needed.

The upcoming v2.0.1 release (Qt) will introduce a COPR repository for even easier updates.

bash — AlmaLinux 9
# 1. Download the RPM from GitHub Releases
# 2. Install with DNF (it handles the rest)
$ sudo dnf install \
  ./almalinux-creative-installer-1.0.14-1.el9.noarch.rpm

# 3. Launch and start creating!
$ almalinux-creative-installer
bash — v2.0.1 + COPR
# Enable the official COPR repository
$ sudo dnf copr enable \
  tristantheroux/almalinux-creative-installer

# One-command installation & updates
$ sudo dnf install almalinux-creative-installer
🚀 COPR support will launch alongside v2.0.1 (The Qt Migration).
